# ians-only-friend
A Discord bot for personal use.

# Installation

```
nvm install v7.8.0
nvm use 7.8.0
npm install --save discord.js
npm install discord.js-commando --save
```
Ignore any `UNMET PEER DEPENDENCY` warnings, those are optional and add support for other things like voice.

# Running the Bot

```
node --harmony .
```
